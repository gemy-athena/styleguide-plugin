#Athena UI Toolkit

## Installation

### Yarn
`` 
yarn add athena-ui-toolkit
``

### Npm
`` 
npm i athena-ui-toolkit
``

## Configuration

Pour utiliser les classes CSS fournit par le toolkit, vous devez importer le fichier global dans votre projet.

```css
@import '~athena-ui-toolkit/scss/ui.scss'
```
